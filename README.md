Temperature Converter

Description:
This program reads city and temps in one temperature and then converts them

The program does the requirements as outlined in the 210 Programming languages course. This is implementing reading a file and using the string tools to change the measurement.

Improvements would be to give a user display for what has been changed.

The most difficult part was learning to clear a register. Most places I looked up just did an arbitrary amount of characters like the next 1000 with a new line character being the thing used to clear the register. I definitely kept using that. 

In general reading and writing to files is very useful across many different structures.
